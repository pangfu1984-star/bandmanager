# BandManager 用户操作手册

## 快速开始

```bash
# 安装依赖并启动
npm install && npm run dev
```

浏览器自动打开 http://localhost:5173，应用首次启动会自动加载示例数据。

---

## 功能模块说明

### 1. 乐队与成员管理（乐队设置）

**功能：**
- 查看/编辑乐队基础信息（仅管理员）
- 添加/编辑/删除成员（仅管理员）
- 切换当前身份（在"我的"页面）

**操作步骤：**
1. 点击侧边栏「乐队设置」
2. 点击「编辑」修改乐队名称和简介
3. 点击「添加成员」填写姓名、乐器、角色
4. 可在「我的」页面底部切换自己的身份（用于演示不同权限）

---

### 2. 日程管理

**功能：**
- 月/周/日视图日历展示
- 创建排练/演出/录音/其他日程
- 自动检测时间冲突
- 附件上传（图片/PDF）

**操作步骤：**
1. 点击侧边栏「日程管理」
2. 点击右上角「新建日程」或直接点击日历上的日期
3. 填写标题（必填）、类型、开始/结束时间
4. 如有时间冲突，系统会弹出警告，确认后可强制保存
5. 点击日历上的日程卡片，可查看详情、编辑或删除

---

### 3. 任务管理

**功能：**
- 按待办/已完成/逾期三个 Tab 分类查看
- 逾期任务顶部横幅提醒 + 标红高亮
- 勾选任务完成状态
- 浏览器推送通知（需授权）

**操作步骤：**
1. 点击侧边栏「任务管理」
2. 点击「新建任务」，填写名称、指派人、截止时间
3. 可关联具体日程（可选）
4. 点击任务左侧圆圈按钮完成/取消完成

---

### 4. 乐谱库

**功能：**
- 上传 PDF / 图片 / 音频乐谱（单文件 ≤10MB，总计 ≤500MB）
- 按歌曲名/乐器/调式/标签模糊搜索
- 列表/网格视图切换
- PDF 预览（前3页）、图片全屏查看、音频播放
- 下载原文件

**操作步骤：**
1. 点击「上传乐谱」，选择文件
2. 填写歌曲名（必填）、调式、乐器、类型、标签
3. 点击乐谱的「预览」按钮在线查看
4. 点击「下载」按钮下载原始文件

---

### 5. 歌单管理

**功能：**
- 创建/删除歌单
- 从乐谱库添加歌曲
- 拖拽调整歌曲顺序（PC端鼠标拖拽，移动端长按）
- 为每首歌添加演奏备注
- 导出 PDF / JSON 格式

**操作步骤：**
1. 点击「新建」创建歌单
2. 进入歌单后，点击「从乐谱库添加」
3. 拖拽左侧「⠿」图标调整歌曲顺序
4. 点击「✏️」图标编辑该曲目的演奏备注
5. 点击「PDF」或「JSON」按钮导出歌单

---

### 6. 数据备份与恢复

**功能：**
- 导出完整 ZIP 备份（含所有文件）
- 导入 ZIP 恢复数据（覆盖当前数据）
- 查看最近3条自动备份，支持一键回滚
- 清空当前乐队所有数据

**操作步骤：**
1. 点击「下载备份 ZIP」导出完整备份
2. 跨设备使用时，点击「选择备份文件」上传 ZIP 恢复
3. 自动备份列表中点击「回滚」可恢复到指定时间点

## 常见问题

**Q: 首次打开没有数据怎么办？**
A: 正常情况下首次启动会自动初始化示例数据（1个乐队、3名成员、2个日程、5个任务、3首乐谱、1个歌单）。

**Q: 数据存在哪里？**
A: 所有数据存储在浏览器的 IndexedDB 中，完全本地化，不会上传任何服务器。

**Q: 如何在多台设备使用？**
A: 先在原设备导出备份 ZIP，在新设备打开应用后导入该备份即可。

**Q: 清空数据后能恢复吗？**
A: 如果有导出备份，可以导入恢复。否则无法恢复，建议定期导出备份。

**Q: PWA 如何安装到桌面？**
A: 在 Chrome/Edge 地址栏右侧会出现「安装」按钮，点击即可安装到桌面。

```bash
# 运行容器
docker run -p 8080:80 bandmanager

# 访问
open http://localhost:8080
```
